## Issue
$ARGUMENTS

Use the `debugger` agent to:
1. Analyze logs and error messages
2. Trace execution through handlers → services → repositories
3. Identify root cause with evidence
4. Provide fix recommendations

## Common Debug Targets
- Firebase Functions logs
- Shopify webhook delivery
- Firestore query performance
- API endpoint failures
- CI/CD pipeline issues (GitHub Actions)
