Use the `shopify-app-tester` agent to:
1. Analyze modified files and functions
2. Map changes to affected areas (backend, frontend, extensions)
3. Generate prioritized testing checklist
4. Highlight risk areas

## Output Structure
- Summary of changes
- Affected regions (backend, frontend, extensions, database)
- Testing checklist (critical, high-impact, regression)
- Risk areas and testing tips
